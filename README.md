- **JavaScript**
- **React**

## How to Run

1. Create a `.env` file in the root directory and add the backend deployed URL:

    ```bash
    REACT_APP_BACKEND=<your_backend_url>
    ```

2. Install dependencies:

    ```bash
    npm ci
    ```

3. Start the application:

    ```bash
    npm run start
    ```
